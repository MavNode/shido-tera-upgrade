# Shido Node Upgrade Script

Automated script for upgrading Shido nodes to Tera.

## Installation

Clone the repository:

```bash
git clone https://github.com/MavNode/shido-tera-upgrade
cd shido-tera-upgrade
```

## Usage

```bash
chmod +x script.sh
./script.sh
```
